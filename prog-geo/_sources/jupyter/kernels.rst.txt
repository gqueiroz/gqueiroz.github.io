Kernels
=======


O ambiente Jupyter permite executarmos múltiplos *kernels*, incluindo diferentes linguagens de programação.


Na distribuição Anaconda é possível criar um novo *kernel* para a linguagem R através do seguinte comando::

    conda install -c r r-essentials
