Removendo Objetos do Banco de Dados
===================================


Removendo Tabelas
-----------------


.. code-block:: sql

    DROP TABLE IF EXISTS nome-tabela;


.. code-block:: sql

    DROP TABLE IF EXISTS nome-tabela CASCADE;


Removendo Schemas
-----------------


.. code-block:: sql

    DROP SCHEMA nome-tabela;


.. code-block:: sql

    DROP SCHEMA IF EXISTS nome-tabela CASCADE;


Removendo Views
---------------


.. code-block:: sql

    DROP VIEW nome-view;
