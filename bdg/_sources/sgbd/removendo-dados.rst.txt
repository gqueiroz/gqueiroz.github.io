.. _cap_sgbd_removendo_dados:

Removendo Dados
===============


Exemplos:


**1.** Eliminando o estudante de código de matrícula ``11``:


.. code-block:: sql

    DELETE FROM estudante WHERE matricula = 11;


**2.** Eliminando a disciplina de ``Bilogia``:


.. code-block:: sql

    DELETE FROM disciplina WHERE titulo = 'Bilogia';
