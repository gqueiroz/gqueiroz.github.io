.. _cap_sgbd_removendo_dados:

Removendo Dados
===============


Exemplos:


**1.** Eliminando o aluno de código de matrícula ``11``:


.. code-block:: sql

    DELETE FROM alunos WHERE matricula = 11;


**2.** Eliminando o curso de ``biologia``:


.. code-block:: sql

    DELETE FROM cursos WHERE titulo = 'biologia';


.. note::

    Repare que nesse último comando ``DELETE``, como havíamos definido uma restrição de integridade referencial entre as tabelas ``alunos`` e ``cursos``, as linhas da tabela ``alunos`` relacionadas ao código do curso de ``biologia`` foram automaticamente eliminadas.
