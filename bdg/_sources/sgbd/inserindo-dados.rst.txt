Inserindo Dados
===============


Sintaxe:


.. code-block:: sql

    INSERT INTO nome-tabela (campo1, campo2, ...)
                     VALUES (v1a, v2a, ...)[,(v1b, v2b, ...)];


Vamos inserir quatro tuplas na tabela ``cursos``:


.. code-block:: sql

    INSERT INTO cursos (titulo, carga_horaria)
                VALUES ('informatica', 4);

    INSERT INTO cursos (titulo, carga_horaria)
                VALUES ('matematica', 6);

    INSERT INTO cursos (titulo, carga_horaria)
                VALUES ('quimica', 2);

    INSERT INTO cursos (titulo, carga_horaria)
                VALUES ('biologia', 3);


Vamos inserir quatro tuplas na tabela ``alunos``:

.. code-block:: sql

    INSERT INTO alunos (nome, curso_id)
                VALUES ('gilberto', 1);

    INSERT INTO alunos (nome, curso_id)
                VALUES ('edson', 2);

    INSERT INTO alunos (nome, curso_id)
                VALUES ('eduardo', 3);

    INSERT INTO alunos (nome, curso_id)
                VALUES ('cassia', 4);


.. note::

    Observe que nos comandos acima relativos à tabela ``cursos``, omitimos os valores para a coluna ``curso_id``. Como definimos essa coluna como ``SERIAL``, o PostgreSQL utiliza valores de uma sequência para preencher este campo quando omitido. De maneira similar, omitimos os valores para a coluna ``curso_id`` nos comandos de inserção na tabela ``alunos``.
