Atualizando Dados
=================


Exemplos:


**1.** Alterando a carga horária de todos os cursos para 60 horas:


.. code-block:: sql

    UPDATE cursos
       SET carga_horaria = 60;


**2.** Alterando o nome do curso ``matematica`` para ``estatistica``:


.. code-block:: sql

    UPDATE cursos
       SET titulo = 'estatistica'
     WHERE titulo = 'matematica';
