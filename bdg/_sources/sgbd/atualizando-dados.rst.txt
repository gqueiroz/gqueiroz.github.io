Atualizando Dados
=================


Exemplos:


**1.** Alterando o número de creditos de todas as disciplinas para 04:


.. code-block:: sql

    UPDATE disciplina
       SET creditos = 4;


**2.** Alterando o nome da disciplina ``Matemática`` para ``Matemática Avançada``:


.. code-block:: sql

    UPDATE disciplina
       SET titulo = 'Matemática'
     WHERE titulo = 'Matemática Avançada';
